About OverTrack

OverTrack was a B2C subscription project I built and have since retired. This site represents a functional snapshot of the game tracking system as it existed when operational.

The system used computer vision to automatically track gameplay statistics for Overwatch, Valorant, and Apex Legends, providing players with detailed match history, statistics, and insights.

This archive preserves the original functionality and interface, using real game data from when the system was live, to demonstrate the capabilities of the platform.
